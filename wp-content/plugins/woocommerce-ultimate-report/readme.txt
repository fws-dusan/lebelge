Introducing WooCommerce Ultimate Reports plug-in, a powerful and advanced repoting tool, comprehensive sales summary on Dashboards, Variation based reports; features like Auto Email summary reports, New/Repeat Customer Analysis, Frequently Order Customers, Graphical Presentation of reports, Stock reports etc.

== Description ==
**Comprehensive features in WooCommerce Ultimate Reports are as follows:**

* Cost of Goods/Profit Report/Analysis
* Sales Trend Analysis Reports
* Enhanced Stock Reports, Stock Summary with Valuation, Stock Email Alerts (Zero Level, Minimum Level)
* New/Repeat Customer Analysis
* Frequently Order Customers
* Customer in Price Point
* Customers who has not purchased within particular date range
* Variation Reports - Variation and Variation Attribute wise filters are available
* Tax Reports - City, State, Country, Zip Code, Tax Name wise summaries
* 8 different Cross Tab Reports
* Projected Vs Actual Sales Report
* Top n Profit earning products
* Product wise Customer List
* Low stock alert on Dashboard and email notification
* Stock Planner and Stock Valuation
* Coupon Discount Type wise reporting
* Product, Category, Tag wise summary
* Various filters available like State, Country, City, Email, Orde ID, Variation, Product, Category, Coupon, Order Status, Postcode (Zip)
* User Access Rights (Set permission for menus & reports, Set Permission to user role.)
* New Customer / Repeat Customer
* Daily Sales Report
* Customer wise New Product
* Date filter on dashboard
* Product wise New Customer

Plugin is compatible with WooCommerce 3.0+ and is tested on 5.2.2 version.

== Installation ==

* Install the plugin from the 'Plugins' section in your dashboard (Go to `Plugins > Add New > Upload Plugin').
* Unzip downloaded plugin and upload it to the plugins folder of your WordPress installation (`wp-content/plugins/` directory of your WordPress installation).
* Activate it through the 'Plugins' section.
* Access the settings page from WordPress backend under WooCommerce Ultimate Reports section.
